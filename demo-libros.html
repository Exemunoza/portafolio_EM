Sistema de Compra de Libros

Aplicación con registro/login y compra protegida con JWT

Descripción del Proyecto

Este sistema permite a los usuarios registrarse e iniciar sesión para acceder al inventario de libros. Solo los usuarios autenticados pueden realizar compras, y el sistema controla la disponibilidad de stock.

  • Problema resuelto: Control de inventario y acceso seguro.
  • Tecnologías utilizadas: Node.js, Express, JWT, Bootstrap.
  • Repositorio: Ver en GitHub
